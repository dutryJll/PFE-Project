✅ CE QUE VOUS DEVEZ FAIRE MAINTENANT:

1. LIRE LES GUIDES (5 min):
   📄 PROCHAINES_ETAPES.txt
   📄 QUICKSTART_5_MINUTES.md

2. OUVRIR 4 TERMINALS PowerShell DIFFÉRENTS

3. COPIER-COLLER CHACUNE DE CES COMMANDES (dans 4 terminals différents):

─────────────────────────────────────────────────────────────────────
TERMINAL 1 - Initialisation & Intégration (exécuter UNE FOIS):
─────────────────────────────────────────────────────────────────────

cd c:\Users\HP\Desktop\PFE
.\integrate_sprint2.bat

💡 Ce script automatise TOUT: dépendances, migrations, tests, etc.
⏱️  Temps: ~15 minutes

⬇️ APRÈS QUE CE SCRIPT SE TERMINE, PASSER AU TERMINAL 2


─────────────────────────────────────────────────────────────────────
TERMINAL 2 - Démarrer Redis (Message Broker):
─────────────────────────────────────────────────────────────────────

docker run -d -p 6379:6379 redis:latest

💡 L'image Docker sera téléchargée la première fois
⏱️  Temps: ~2 minutes (ou immédiat si Docker est installé)

Vérification:
  redis-cli ping
  >>> PONG (succès!)


─────────────────────────────────────────────────────────────────────
TERMINAL 3 - Démarrer Celery Worker (Traitement OCR):
─────────────────────────────────────────────────────────────────────

cd c:\Users\HP\Desktop\PFE\isimm-platform\services\candidature_service
celery -A candidature_service worker -l info

💡 Doit afficher: "[...] Ready to accept tasks"
⚠️  NE PAS QUITter ce terminal (laisser actif)


─────────────────────────────────────────────────────────────────────
TERMINAL 4 - Démarrer Celery Beat (Scheduler):
─────────────────────────────────────────────────────────────────────

cd c:\Users\HP\Desktop\PFE\isimm-platform\services\candidature_service
celery -A candidature_service beat -l info

💡 Doit afficher: "beat: Scheduler started"
⚠️  NE PAS QUITter ce terminal (laisser actif)


─────────────────────────────────────────────────────────────────────
TERMINAL 5 - Démarrer Django Server:
─────────────────────────────────────────────────────────────────────

cd c:\Users\HP\Desktop\PFE\isimm-platform\services\candidature_service
python manage.py runserver 8003

💡 Doit afficher: "Starting development server at http://127.0.0.1:8003/"
⚠️  NE PAS QUITter ce terminal (laisser actif)
