## 🔗 COMMANDES UTILES À RETENIR

### Voir tous les endpoints:
```powershell
cd candidature_service
python manage.py show_urls | grep dossier
```

### Créer un token d'authentification:
```powershell
python manage.py shell
```

```python
from rest_framework.authtoken.models import Token
from django.contrib.auth import get_user_model

User = get_user_model()
admin = User.objects.get(username='admin')
token = Token.objects.get_or_create(user=admin)[0]
print(f"Token: {token.key}")
```

### Tester un endpoint:
```powershell
$token = "YOUR_TOKEN_HERE"
$headers = @{ "Authorization" = "Token $token" }
Invoke-WebRequest -Uri "http://localhost:8003/api/dossier/types/1/" `
  -Headers $headers | ConvertFrom-Json
```

### Voir les tests:
```powershell
python -m pytest candidature_app/tests.py -v
```

## 🆘 SI VOUS ÊTES BLOQUÉ

### Erreur: "Redis connection refused"
```powershell
# Solution 1: Docker
docker run -d -p 6379:6379 redis:latest

# Solution 2: Vérifier Redis en local
redis-cli ping
```

### Erreur: "ModuleNotFoundError"
```powershell
pip install -r requirements_depot_dossier.txt --force-reinstall
```

### Erreur: "Port 8003 already in use"
```powershell
python manage.py runserver 8004
```

### Erreur: "Migrations not applied"
```powershell
python manage.py migrate candidature_app
```
